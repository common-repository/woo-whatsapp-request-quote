Easily add Whatsapp link to request quote in woocommerce product.

== Description ==
Whatsapp for woocommerce lets you add whatsapp button/link into your
woocommerce product pages to get quote request.
Users will be able to get in touch with you via whatsapp and the product
page name and link will be added to their chat.
You can change the default message through the settings and select icons
from the library.

Please give a 5* rating if it does what you are expecting OR request customization via support.

== Installation ==
- Upload zip file.
- Activate the plugin.
- Go to settings option and do the necessary changes.
- No shortcode needed.
- Done.
